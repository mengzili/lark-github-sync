Lark–GitHub Sync

Fill in your credentials, click Deploy. No CLI, no git, no manual config.

First time? Create a Lark app (takes ~3 min)

Open the developer console:

Feishu Console Lark Console
  1. Click Create Custom App (创建企业自建应用)
    GitHub Sync Bot
    Syncs GitHub org members, repos, and event notifications with Lark
  2. Sidebar→Add Capabilities(添加应用能力)→Bot(机器人)
    Click + Add on the Bot card.
  3. Sidebar→Permissions & Scopes(权限管理)
    Search and activate these 6 permissions: contact:user.base:readonly, contact:department.base, contact:contact:readonly_as_app, contact:contact, im:chat, im:message:send_as_bot. Then click Batch Activate (批量开通).
  4. Sidebar→Version Management & Release(版本管理与发布)
    Click Create a version (创建版本) → set Availability to All employees (全部员工) → Submit for release (申请发布). If you are the Lark admin, approve it in Admin Console (管理后台) → App Review (应用审核).
  5. Sidebar→Credentials & Basic Info(凭证与基础信息)
    Copy the App ID and App Secret and paste them below.
