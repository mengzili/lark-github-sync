How to create a Lark app (~3 min)
Feishu Console Lark Console
  1. Click Create Custom App (创建企业自建应用)
    GitHub Sync Bot
    Syncs GitHub org members, repos, and event notifications with Lark
  2. Sidebar→Add Capabilities(添加应用能力)→Bot(机器人)
    Click + Add on the Bot card.
  3. Sidebar→Permissions & Scopes(权限管理)
    Search and activate: contact:user.base:readonly, contact:user.id:readonly, contact:department.base:readonly, contact:contact.base:readonly, im:chat, im:message:send_as_bot. Click Batch Activate (批量开通).
  4. Sidebar→Version Management(版本管理与发布)
    Create a version (创建版本) → set Availability to All employees (全部员工) → Submit (申请发布). If you are the admin, approve in Admin Console (管理后台) → App Review.
  5. Sidebar→Credentials & Basic Info(凭证与基础信息)
    Copy App ID and App Secret below.

0 = all employees.
