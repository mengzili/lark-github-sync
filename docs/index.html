Lark–GitHub Sync

Set up in ~5 minutes. Everything happens in the browser — nothing to install.

1
Use this template ~10 sec

Create a private copy in your GitHub org. The name, description, and visibility are pre-filled — just pick your org and click Create repository.

Use this template →

Once created, paste your new repo URL here — the links below will update automatically:

2
Create a Lark app ~2 min

Open the developer console, click Create Custom App, and paste these pre-filled values:

GitHub Sync Bot
Syncs GitHub org members, repos, and event notifications with Lark
Feishu Console → Lark Console →

After creating the app, go to Permissions & Scopes and add:

PermissionPurpose
contact:user.base:readonlyLook up users by email
contact:department.baseRead departments
contact:contactManage contacts
im:chatCreate group chats
im:message:send_as_botSend notifications

Submit for admin approval, then set App Availability → All employees.

Copy the App ID and App Secret — you'll paste them in the next step.

3
Set 3 secrets on your repo ~1 min

Go to your repo's Settings → Secrets and variables → Actions → New repository secret and add:

Secret nameValue
LARK_APP_IDFrom the Lark app you just created
LARK_APP_SECRETFrom the Lark app you just created
SYNC_GITHUB_TOKENA GitHub PAT — click below to create one with pre-filled scopes
Open repo Secrets → Create GitHub Token →
Tip: The "Create GitHub Token" link pre-fills the name and scopes. Just click Generate token and paste it as SYNC_GITHUB_TOKEN.
4
Run Initial Setup ~1 min

Go to Actions → 🚀 Initial Setup → Run workflow. Fill in:

InputWhat to enter
github_orgYour GitHub organization name
lark_domainfeishu (China) or lark (international)
lark_department_id0 for all employees, or a specific department ID
Open Actions tab →

The workflow will configure everything, sync members, create group chats, and deploy notifications.

✓ You're done!

Everything syncs automatically every day at 02:00 UTC.
New Lark department members get GitHub access automatically.
Every repo push, PR, and issue sends a Lark notification.
